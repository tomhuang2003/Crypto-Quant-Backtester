# Applied Data Science Portfolio

This repository contains real-world data science projects focused on real estate market analysis and predictive modeling, developed as part of the **WorldQuant University Applied Data Science Lab**.

## 🇲🇽 Mexico City Apartment Price Predictor
- **Objective**: Develop a machine learning model to estimate apartment prices in Mexico City.
- **Methodology**: 
  - Engineered a robust `wrangle` function to clean data, handle missing values, and remove outliers using **quantile-based filtering**.
  - Built a predictive pipeline using **One-Hot Encoding** for categorical data (Boroughs) and **Ridge Regression** to minimize overfitting.
  - Implemented an interactive **Streamlit** dashboard for real-time price predictions.
- **Tech Stack**: Python, Scikit-Learn, Pandas, Streamlit.

## 🇧🇷 Brazil Housing Market Analysis
- **Objective**: Conduct an Exploratory Data Analysis (EDA) of the Brazilian real estate market across different regions.
- **Methodology**: 
  - Performed data integration of multiple CSV sources with currency conversion (BRL to USD).
  - Analyzed the correlation between home size and price across different states (e.g., Rio Grande do Sul).
  - Created geospatial visualizations using **Plotly Express Mapbox** to identify regional price hotspots.
- **Tech Stack**: Pandas, Plotly Express, Matplotlib.
